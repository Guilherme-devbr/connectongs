GeoApify API

O que é a GeoApify?

A GeoApify é uma API de geolocalização e mapas utilizada para localizar instituições e exibir informações geográficas de forma precisa dentro da plataforma Connect Ong.

Objetivo no Projeto

  • Localizar ONGs próximas ao usuário.
  • Permitir busca por instituições distantes.
  • Exibir endereços de forma organizada.
  • Facilitar a conexão entre doadores e ONGs.

Como a API funciona?

A GeoApify utiliza dados de localização do dispositivo para identificar a posição do usuário e comparar com os endereços das instituições cadastradas na plataforma.

Benefícios para o Connect Ong

  • Maior precisão na localização.
  • Experiência mais intuitiva.
  • Facilidade para encontrar instituições.
  • Maior transparência para os doadores.
  • Conexão rápida entre usuários e ONGs.

Importância para a Transparência

Um dos pilares do Connect Ong é a confiança. Com a GeoApify, o usuário consegue visualizar a localização das instituições, aumentando a segurança na hora de realizar uma doação ou participar como voluntário.

Integração com o Cadastro das ONGs

As informações cadastradas pelos responsáveis das instituições alimentam a base de localização utilizada pela GeoApify, permitindo que a ONG seja encontrada através da busca por proximidade.
